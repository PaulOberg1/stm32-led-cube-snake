# STM32 3D LED Cube Snake

Bare-metal firmware implementing a real-time 3D Snake game on an STM32-driven LED cube.

## Features

- ADC joystick input
- UART LED cube driver
- 3D coordinate game engine
- Collision detection
- Wrap-around movement
- libopencm3 bare-metal firmware

## Hardware

- STM32 microcontroller
- 8x8x8 LED cube
- Analog joystick
